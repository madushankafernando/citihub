Cloud Platform Services

Architecture, design and hands-on engineering of secure and scalable private and public cloud platforms

What We Do?

As the scale of cloud adoption increases and the breadth of services begins to grow, Citihub is helping financial institutions build self-service mechanisms for the automated delivery of secure application environments. These environments facilitate access to a heterogeneous set of services and service providers, and handle the rapid feature velocity of the cloud in a scalable and secure way.

For applications that aren’t yet ready to make the leap into public cloud, we work alongside clients and vendor organizations to create a public cloud-like experience. This provides modern platforms for applications to modernize over time and reduces the friction of migrating to public cloud when the time comes.

Our Capabilities

Public cloud enterprise foundations

Implement and integrate the core components necessary for complex, regulated firms to safely operate in the public cloud including networking, identity, encryption, logging and monitoring.

Application environment delivery

Automate the delivery of self-service, secure application environments in Azure, AWS and GCP to support access to the broad range of services on offer.

Streamlined service enablement

Streamline the process for enabling developer access to new services, by rapidly assessing new services and securely integrating them into the public cloud application environment.

Kubernetes platform

Design and build a cloud native stack for modern applications and workload portability, based on the broad sets of tools and integrations available in the Kubernetes ecosystem.

Container lifecycle management

Bake consistent and appropriate controls into the lifecycle management of container-based applications – from sourcing, building, distribution, deployment and runtime to identification and revocation of vulnerable images.

Hybrid cloud data access

Facilitate secure and controlled communication between services running in public cloud and services running on-premise with the dynamism required for modern applications to operate.
